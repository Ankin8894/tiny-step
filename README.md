# Tiny Step 💬

A simple, mobile-friendly chatbot interface built with HTML, CSS, and JavaScript.
The project simulates a conversation between a user and a bot using preset buttons for responses.

This project was created as a beginner-friendly web development exercise and demonstrates
how a small interactive web app can be built using only front-end technologies.

## 📱 Features
- Simple chatbot-style interface
- User and bot message bubbles
- Button-based responses
- Mobile-friendly layout
- Scrollable chat area
- Clean, minimal design

## 🛠 Technologies Used
**HTML** – page structure
**CSS** – layout and styling
**JavaScript** – interaction and chat behavior

No frameworks or external libraries are used.

## 📂 Project Structure
tiny-step/
│
├── index.html      # Main webpage
├── style.css       # Styling and layout
└── script.js       # Chat interaction logic

## 🚀 How to Run the Project
**Option 1: Run Locally**
- Download or clone the repository.
- Make sure these files are in the same folder:
index.html
style.css
script.js
- Open index.html in any web browser.

**Option 2: Run on Mobile**
- Upload the project to GitHub.
- Enable GitHub Pages in repository settings.
- Open the generated link on your phone.
- Add it to your home screen for an app-like experience.
